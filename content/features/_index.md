# Features

Antonio's Pizza Pub Inventory System provides comprehensive inventory management capabilities.

## Core Features

### ✅ Real-time Inventory Management
Add, update, and remove items instantly. All changes are saved immediately to the database and synced across devices.

### ✅ Low Stock Alerts
Automatic warnings when items fall below the threshold (3 units). Low stock items are highlighted with a red warning card on the home page.

### ✅ Search Functionality
Quickly find items by name with a real-time, case-insensitive search feature. Results update as you type.

### ✅ Multiple Sort Options
Organize your inventory with various sorting options:
- Name A–Z / Z–A
- Quantity Low→High / High→Low
- Low Stock First

### ✅ User-Friendly Interface
Intuitive design with clear navigation and visual feedback. The interface is designed to be easy to use for restaurant staff.

### ✅ Remote Access
Access inventory from anywhere when PocketBase is accessible. Perfect for checking inventory from home or while away from the restaurant.

### ✅ Cross-Platform Support
Works on multiple platforms:
- Windows Desktop
- macOS Desktop
- Linux Desktop
- Android (phones and tablets)
- iOS (iPhone and iPad)
- Web Browsers

### ✅ Offline Capability
The app structure supports offline capabilities (requires PocketBase for data synchronization).

## Technical Features

- Built with Flutter for cross-platform compatibility
- PocketBase backend for reliable data storage
- Real-time synchronization
- Responsive design
- Error handling and user feedback

# Getting Started

Welcome to Antonio's Pizza Pub Inventory System! This guide will help you get up and running quickly.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Flutter SDK** (version 3.9.2 or higher)
  - Installation guide: [https://docs.flutter.dev/get-started/install](https://docs.flutter.dev/get-started/install)
  - Verify installation: `flutter --version`
  
- **PocketBase** (latest version)
  - Download from: [https://pocketbase.io/docs/](https://pocketbase.io/docs/)
  
- **Dart SDK** (included with Flutter)

- **An IDE** (VS Code, Android Studio, or IntelliJ IDEA with Flutter plugins)

## Quick Setup Steps

1. [Install PocketBase](/setup/installing-pocketbase/)
2. [Run PocketBase](/setup/running-pocketbase/)
3. [Configure the Database](/setup/configuring-pocketbase/)
4. [Install Dependencies](/getting-started/installing-dependencies/)
5. [Run the Application](/getting-started/running-the-application/)

## Next Steps

Once you have the application running, check out the [Usage Guide](/usage/) to learn how to manage your inventory.

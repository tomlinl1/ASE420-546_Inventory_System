## Quick Checks

Before troubleshooting, verify:
- PocketBase is running on port 8090
- You're in the correct directory (`inventory_app`)
- All dependencies are installed (`flutter pub get`)
- Flutter installation is correct (`flutter doctor`)

## Still Need Help?

If you're still experiencing issues after trying the solutions, check:
- PocketBase logs for backend errors
- Flutter console output for app errors
- Network connectivity
- Firewall settings

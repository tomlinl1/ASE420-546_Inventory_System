# Using the Application

This guide covers all the features and functionality of the inventory system.

## Main Features

1. [Home Page](/usage/home-page/) - Overview and navigation
2. [Checking Stock](/usage/checking-stock/) - View and search inventory
3. [Adding Stock Items](/usage/adding-stock/) - Create new inventory items
4. [Updating Stock Items](/usage/updating-stock/) - Modify existing items
5. [Removing Stock Items](/usage/removing-stock/) - Delete items from inventory

## Quick Reference

- **Home:** Main dashboard with low stock alerts
- **Check Stock:** View, search, sort, and update quantities
- **Update Stock:** Add, edit, or delete inventory items

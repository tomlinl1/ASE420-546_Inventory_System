# Adding Stock Items

Learn how to add new items to your inventory.

## Navigation

From the home page, tap the **"Update Stock"** button to access this page.

## Steps to Add an Item

### 1. Fill in the Form

Three required fields:

- **Item Name:** Enter the name of the inventory item
  - Example: "Mozzarella Cheese"
  
- **Quantity:** Enter the current quantity as a number
  - Example: "10"
  - Must be a valid integer (whole number)
  
- **Unit:** Enter the unit of measurement
  - Example: "lbs", "gal", "boxes", "oz"
  - Common units: pounds, gallons, boxes, ounces, kilograms

### 2. Save the Item

1. Tap the **"Add Stock"** button
2. A green success message will appear confirming the item was added
3. The form will clear automatically
4. The new item will appear in the list below

### 3. View All Items

Scroll down to see all inventory items in a list. Each item shows:
- Name
- Quantity
- Unit

## Example

To add "Tomato Sauce" with 5 gallons:

1. Item Name: `Tomato Sauce`
2. Quantity: `5`
3. Unit: `gal`
4. Tap "Add Stock"

## Tips

- Item names are case-sensitive in searches
- Use consistent unit names (e.g., always use "lbs" not "pounds")
- Quantity must be 0 or greater

# Checking Stock

The Check Stock page allows you to view, search, sort, and update inventory quantities.

## Navigation

From the home page, tap the **"Check Stock"** button to access this page.

## Features

### Search Bar

- Located at the top of the page
- Type to filter items by name
- Search is **case-insensitive**
- Real-time filtering as you type

### Sort Options

Tap the sort icon (☰) in the top right to access sorting options:

- **Name A–Z:** Sort alphabetically ascending
- **Name Z–A:** Sort alphabetically descending
- **Quantity Low → High:** Sort by quantity ascending
- **Quantity High → Low:** Sort by quantity descending
- **Low Stock First:** Prioritize items with low quantities

### Update Quantities

Each item displays:
- Item name
- Current quantity
- Unit of measurement

To update quantities:
- Use the **"-" (minus) button** to decrease quantity by 1
- Use the **"+" (plus) button** to increase quantity by 1
- Quantity cannot go below 0
- A loading indicator appears while the update is processing
- Success/error messages appear at the bottom of the screen

### Navigation

Tap the back arrow (←) in the top left to return to the home page.

## Quick Tips

- Search works in real-time - no need to press enter
- Use "Low Stock First" sorting to quickly identify items that need restocking
- Quantity updates save immediately to the database

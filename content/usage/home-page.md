# Home Page

The home page is the main entry point of the application. It provides an overview of your inventory and quick access to all features.

## Features

### Welcome Message
Displays "Welcome to the Inventory System!" at the top of the page.

### Navigation Buttons
Three main action buttons:

- **Check Stock:** View and manage current inventory with search and sort options
- **Update Stock:** Add, edit, or delete inventory items
- **Website:** Opens Antonio's Pizza Pub website in your browser

### Low Stock Alert
If any items have quantity below 3, a red warning card will appear showing:

- Number of low stock items
- A "View" button to navigate to the low stock items

### Low Stock Items List
Displays up to 5 low stock items with their current quantities and units. If there are more than 5 items, a "View all" button appears.

## Refreshing Data

Pull down on the home page to refresh inventory data and get the latest information from the database.

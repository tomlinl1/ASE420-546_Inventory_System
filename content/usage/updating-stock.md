# Updating Stock Items

Learn how to modify existing inventory items.

## Navigation

From the home page, tap the **"Update Stock"** button to access this page.

## Steps to Update an Item

### 1. Find the Item

Scroll through the list of items to find the one you want to edit.

### 2. Edit the Item

1. Tap the **edit icon (✏️)** on the right side of the item
2. The form fields will be automatically filled with the current item data
3. The button will change from "Add Stock" to "Update Stock"

### 3. Modify the Fields

Change any of the fields as needed:
- Item Name
- Quantity
- Unit

You can modify one or all fields.

### 4. Save Changes

1. Tap the **"Update Stock"** button
2. A green success message will confirm the update
3. The form will clear
4. The item list will refresh to show the updated data

### 5. Cancel Editing

After tapping edit, you can:
- Clear the form fields manually
- The button will revert to "Add Stock" when fields are empty

## Example

To update "Mozzarella Cheese" quantity from 5 to 15:

1. Find "Mozzarella Cheese" in the list
2. Tap the edit icon (✏️)
3. Change Quantity from `5` to `15`
4. Tap "Update Stock"

## Tips

- You can update just the quantity or all fields at once
- Changes are saved immediately to the database
- The list refreshes automatically after updating

## Alternative Method

You can also quickly update quantities from the [Check Stock](/usage/checking-stock/) page using the + / - buttons without going to the Update Stock page.
